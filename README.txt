Os scripts tem a funcionabilidade de mostrar a cotação do Bitcoin em 3 moedas diferentes sendo elas BRL, USD, EUR usando a API e armazenalas no arquivo exel para visualização
Antes de executar os scripts instale as bibliotecas necessarias atravez do comando "pip install requiriments.txt"
Para executar o script abra o cmd e digite "python cotacao1.py" ou "python cotacao1.py".
Funcionabilidade dos scripts:
cotacao1 : Executa 30 vezes o codigo a cada 60 segundos para adquirir a cotação do BTC nas moedas BRL,USD,EUR.
cotacao2 : Executa a cada 60 segundos para adquirir a cotação do BTC nas moedas BRL,USD,EUR em tempo real atraves da API.
